# Frontend

Next.js App Router com TypeScript, TailwindCSS, Recharts, Framer Motion, Zod e lucide-react.

```bash
npm install
npm run seed
npm run dev
```

Rotas de API internas:

- `POST /api/exams`: gera uma prova com 30 questões.
- `POST /api/exams/submit`: corrige a prova e retorna revisão completa.

O histórico da primeira versão fica em `localStorage`.
